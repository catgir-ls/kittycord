> ### Setup ✨
### Cloning the git repository
```
$ git clone https://github.com/catgir-ls/kittycord.git
$ cd kittycord
```
### Building 
```
$ ./gradlew shadowJar
```
### Running
```
$ mv lib/build/libs/lib-all.jar $MC_SERVER_ROOT/plugins/KittyCord.jar
```
### Configuring
```
$ vim $MC_SERVER_ROOT/plugins/KittyCord/config.yml
```
